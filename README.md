```
✓ Install FFmpeg: apt-get install ffmpeg
✓ Verify STRING_SESSION is valid
✓ Check bot can access voice chat (permissions)
✓ Ensure firewall isn't blocking connections
✓ Try clearing cache: /clearqueue
✓ Restart bot: /restart
→ Still not working? Get help: @axlomm
```

```
✓ Check MONGO_DB_URI format is correct
✓ Add IP address to MongoDB Atlas whitelist
✓ Test locally: mongosh your_connection_string
✓ For Railway: Use MONGO_URL instead of MONGO_DB_URI
✓ Check internet connection
→ Need help? Check: https://t.me/axlomm
```

### **❌ Songs Not Playing?**
```
✓ Check if YouTube/Spotify is accessible in your region
✓ Verify API keys are valid
✓ Ensure URL is correct
✓ Check file size (must be < 2GB)
✓ Try different song/platform
✓ Update yt-dlp: pip install -U yt-dlp
→ Contact support: @axlomm
```
